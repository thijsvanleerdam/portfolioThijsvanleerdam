Smart Connection

HTML

Het portfolio dat moet worden gepresenteerd en wat u nu aan het bekijken bent is zichtbaat op internet. Om dit voor elkaar te krijgen hebben we tijdens de lessen van Smart connectie gewerkt met verschillende opdrachten om HTML te leren kennen.

De eerste website ging aan de hand van een tutorial op YouTube waaruit je een hele basale website krijgt. Ik heb deze opdracht gebruikt om verder te bouwen aan het portfolio wat u nu voor u heeft. Kort om ik heb doormiddel van de tutorial een framework opgezet waarna ik met verder onderzoek op YouTube en W3schools ben gaan doen om tot op dit punt te komen.

CSS

Een website met alleen maar HTML was in de jaren negentig van de vorige eeuw nog geschrikt om informatie over te brengen, maar tegenwoordig voldoet dit niet meer aan de eisen van de gebruikers. Dat wil zeggen dat mensen tegenwoordig meer op het visuele afgaan bij het gebruik van een website. Hiervoor heb ik op deze site dan ook CSS gebruikt voor de opmaak. Delen van de het CSS bestand heb ik zelf geschreven maar ik heb ook gebruik gemaakt van bepaalde Templates die ik op internet heb gevonden.

Machine learning

Verder heb ik tijdens het project wat ik heb gedaan veel gewerkt met Linux en dan specifiek Ubuntu. Tijdens het project zijn we bezig geweest met machine learning om onkruid te herkennen bij de teelt van blauwebessen.

De machinelearing ging via Tensorflow. Tensorflow is een open-source machinelearing tool van Google die gebruikt kan worden voor productie en onderzoek. Met Tensorflow kan je als beginner maar ook als expert API’s krijgen waarmee machineleanring mogelijk wordt gemaakt.

Tensorflow maakt gebruik van Python en hierdoor heb ik ook veel ervaring kunnen opbouwen hiermee. Helaas is het niet helemaal gelukt maar daar zal ik verder nog op ingaan bij het Project. De volgende code heb ik gebruikt tijdens een test run voor tensorflow:

“python train.py --train_dir=/tensorflow/models/research/object_detection/training --pipeline_config_path=ssd_mobilenet_v1_pets.config”
